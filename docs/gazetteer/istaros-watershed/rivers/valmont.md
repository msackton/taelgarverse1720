# Valmont
>[!info]+ Information
> 
>> 

The Valmont flows south of [Addermarch](<../../greater-sembara/addermarch/addermarch.md>) and, with the [Ardonne](<./ardonne.md>), is one of the primary inflows to [Lake Aeulian](<./lake-aeulian.md>).
